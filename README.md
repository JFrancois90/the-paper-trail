# The Paper Trail

Taking politics out of politics.

A non-partisan political fact-checking platform that checks whether politicians' numbers match their own cited sources.

## Stack

- Next.js (App Router)
- TypeScript
- Deployed on Vercel

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Push to GitHub. Vercel builds automatically from the main branch.
